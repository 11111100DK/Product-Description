PlayTogether

là một phần mềm được viết bằng python, giúp người dùng có thể dễ dàng tải về và quản lý các video từ Youtube và bài hát từ Spotify, một cách tiện lợi và đơn giản

Các chức năng của phần mềm

Giao diện trang chủ

Gồm :
+Logo Play Together
+Nút chọn YouTube Extension
+Nút chọn Spotify Extension
+Member Credit

Chức năng :
+Truy cập Spotify Extension
+Truy cập YouTube Extension

Giao diện Youtube

Gồm :
+Giao diện website YouTube
+Nút "Next Page"
+Nút "Back Page"
+Nút "Download Video"
+Nút truy cập quản lý Video tải xuống

Chức năng :
+Trải nghiệm xem và chọn video hoàn toàn trên YouTube
+Di chuyển trở về trang trước và trở lại trang sau
+Tải video về máy (có thể chọn theo độ phân giải)
+Truy cập quản lý video tải xuống

Giao diện Spotify

Gồm :
+List nhạc từ Spotify
+Thanh tìm kiếm bài hát
+Nút "Download"
+Nút truy cập quản lý bài hát tải xuống
+Nút "Go to Spotify"
Chức năng :
+Trải nghiệm nghe bài hát tùy chọn
+Tìm kiếm bài hát theo yêu cầu
+Tải bài hát về máy
+Truy cập quản lý bài hát tải xuống

Giao diện phát nhạc

Thanh phát nhạc gồm :
+Hình ảnh bài hát
+Tên bài hát
+Tên nghệ sĩ
+Nút play, nút next, nút previous
+Thanh thời lượng

Chức năng :
+Phát, dừng, điều chỉnh tùy thích

Giao diện quản lý thư mục tải xuống

Gồm :
+Cây thư mục các folder, file .mp3 và .mp4 đã tải về
+Nút "New Folder"
+Nút "Move File"

Chức năng :
+Chọn file .mp3 và .mp4 bất kì để phát trên trình play
+Di chuyển file theo ý muốn
+Tạo thêm folder
+Chỉ hiện thị các folder và file .mp3, .mp4

Giao diện tạo thư mục

Gồm:
+Thanh nhập tên thư mục mới

Chức năng :
+Tạo thư mục mới theo tên

Giao diện di chuyển thư mục

Gồm :
+Thanh nhập đường dẫn đi chuyển cho thư mục

Chức năng :
+Di chuyển file theo ý muốn

Hướng dẫn cài đặt


Bước 1 : Tải Source Code GitHub.


Bước 2 : Cài đặt môi trường Python theo hướng dẫn.


Bước 3 : Cài đặt các thư viện cần thiết :


+PyQt5:

pip install PyQt5

+youtube_dl:

pip install youtube_dl

+spotipy:

pip install spotipy

+yt_dlp:

pip install yt-dlp

+pytube:

pip install pytube

+tkinter:

pip install tk

Bước 4 : Chạy file MainMenu.py
