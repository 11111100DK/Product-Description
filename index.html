Play Together

Product Descriptions

Giao diện trang chủ

Gồm :
+Logo Play Together
+Nút chọn YouTube Extension
+Nút chọn Spotify Extension
+Member Credit

Chức năng :
+Truy cập Spotify Extension
+Truy cập YouTube Extension

Giao diện Youtube

Gồm :
+Giao diện website YouTube
+Nút "Next Page"
+Nút "Back Page"
+Nút "Download Video"
+Nút truy cập quản lý Video tải xuống

Chức năng :
+Trải nghiệm xem và chọn video hoàn toàn trên YouTube
+Di chuyển trở về trang trước và trở lại trang sau
+Tải video về máy (có thể chọn theo độ phân giải)
+Truy cập quản lý video tải xuống

Giao diện Spotify

Gồm :
+List nhạc từ Spotify
+Thanh tìm kiếm bài hát
+Thanh phát nhạc gồm : hình ảnh bài hát, tên bài hát, tên ca sĩ, nút play, nút next, nút previous, thanh thời lượng

Chức năng :
+Trải nghiệm nghe bài hát tùy chọn
+Tìm kiếm bài hát theo yêu cầu
+Tải bài hát về máy
+Truy cập quản lý bài hát tải xuống

Giao diện quản lý thư mục tải xuống

Gồm :
+Cây thư mục các folder, file .mp3 và .mp4 đã tải về
+Nút "New Folder"
+Nút "Move File"

Chức năng :
+Chọn file .mp3 và .mp4 bất kì để phát trên trình play
+Di chuyển file theo ý muốn
+Tạo thêm folder
+Chỉ hiện thị các folder và file .mp3, .mp4
